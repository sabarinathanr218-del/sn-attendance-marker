# 📋 SN Attendance Marker

Smart Attendance Management App for College Class Representatives.
Built & Designed by **SN**.

## Live App
🌐 [sn-attendance-marker.vercel.app](https://sn-attendance-marker.vercel.app)

## Demo Login
- Username: `SN_founder`
- Password: `SN@admin1`

## Features
- Mark attendance per hour (P / A / OD)
- Add, edit, delete students
- Forgot password reset
- Remember Me login
- Founder dashboard for all classes
- Works offline (PWA)
- Installable as mobile app

## Built With
React + Vite + LocalStorage + Vercel
